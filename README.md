## Things I've Built

**RoboBoat — Autonomous Surface Vessel Hardware Integration**

> Final project involving hardware integration, power-management architecture, KiCad schematics, component selection, soldering, bring-up, and system-level debugging for an autonomous surface vessel.

**4-bit ALU — Logic-Gate-Level VLSI Design Flow**

> Implemented a 4-bit ALU from logic-gate-level schematic design to physical layout validation in Cadence Virtuoso, including DRC/LVS checks and post-layout simulation.

**Custom CPU Toolchain**

> Developed a C-based assembler and ISA simulator for a custom processor architecture, including instruction decoding, register handling, memory-mapped I/O, and interrupt support.

## Currently Building

**TinyMAC — RTL-to-GDSII Physical Design Project**

> A small MAC/DSP accelerator implemented through an open-source ASIC physical design flow using Verilog, OpenLane/OpenROAD, and the SKY130 PDK.
> Goal: study RTL design, synthesis, floorplanning, placement, CTS, routing, timing analysis, area/power tradeoffs, and final GDSII generation.
